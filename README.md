## Getting Started

### Prerequisites

- Flutter SDK (Latest Version)
- Android Studio / VS Code
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/[YourUsername]/pricepals.git
```

2. Navigate to project directory
```bash
cd pricepals
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
